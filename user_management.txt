2. Risk Scoring Engine
[NEW] 
riskEngine.js
Computes 0–100 risk score on registration based on:

Signal	Points	Reason
No referral code	+20	Unvetted signup
Disposable email domain	+25	throwaway email (mailinator, guerrillamail, etc.)
Generic email provider (gmail, outlook)	+10	Not an org email
No department provided	+15	Missing context
No title provided	+10	Missing context
Vague reason (<10 chars)	+15	Low-effort request
Reason contains suspicious keywords	+15	"test", "just looking", etc.
Valid referral code	-20	Vetted by existing user
Color coding: 0–30 🟢 Low, 31–60 🟡 Medium, 61–100 🔴 High
